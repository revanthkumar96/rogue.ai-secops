## Icon Usage Guidelines

1. **Consistency**: Always use the same icons for the same operations
2. **Color Meaning**: Green = success, Red = error, Orange = warning, Blue = info
3. **Flower Motif**: Incorporate 🌸 in success messages
4. **Hand Motif**: Use hand illustrations for parallel/multiple operations
5. **Clarity**: Icons should be recognizable even at small sizes
6. **Accessibility**: Don't rely solely on color; use shapes and symbols
